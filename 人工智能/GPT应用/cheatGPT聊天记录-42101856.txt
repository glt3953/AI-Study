Q: 用Objective-C实现一个快速排序算法

以下是Objective-C实现的快速排序算法代码示例：

```
@implementation NSArray (QuickSort)

- (NSArray *)quickSortedArray {
    if (self.count <= 1) {
        return self;
    }
    
    NSInteger pivotIndex = self.count / 2;
    NSInteger pivotValue = [[self objectAtIndex:pivotIndex] integerValue];

    NSMutableArray *lessArray = [NSMutableArray array];
    NSMutableArray *moreArray = [NSMutableArray array];

    for (int i = 0; i < self.count; i++) {
        NSInteger value = [[self objectAtIndex:i] integerValue];
        if (i == pivotIndex) {
            continue;
        }
        if (value <= pivotValue) {
            [lessArray addObject:@(value)];
        } else {
            [moreArray addObject:@(value)];
        }
    }
    
    NSArray *lessSortedArray = [lessArray quickSortedArray];
    NSArray *moreSortedArray = [moreArray quickSortedArray];
    
    NSMutableArray *resultArray = [NSMutableArray array];
    [resultArray addObjectsFromArray:lessSortedArray];
    [resultArray addObject:@(pivotValue)];
    [resultArray addObjectsFromArray:moreSortedArray];
    
    return [NSArray arrayWithArray:resultArray];
}

@end
```

这个示例使用category扩展了NSArray类，并在其中添加了一个`quickSortedArray`方法来进行快速排序。该方法算法采用了递归思想，将数组分为比中间值小和比中间值大两部分，并对这两部分再次进行快速排序。最后再将排序后的结果合并起来，返回一个有序的数组。
